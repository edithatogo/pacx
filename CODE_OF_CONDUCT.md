# Code of Conduct

PACX follows the principles of the Contributor Covenant 2.1.

## Our standards

We expect everyone participating in this project to:

- be respectful and constructive
- disagree without personal attacks
- avoid harassment, discrimination, and exclusionary behavior
- keep discussion focused on the work
- respect confidentiality when a report is made privately

## Unacceptable behavior

Examples of unacceptable behavior include:

- insulting or threatening language
- repeated disruption of discussion
- personal attacks or harassment
- discriminatory comments or slurs
- doxxing or other privacy violations
- sexualized or abusive language

## Enforcement

Maintainers may:

- remove content
- request that behavior stop
- limit participation in discussions
- restrict access to the project if needed

## Reporting

Report conduct issues to the repository maintainers through a private channel. For urgent or sensitive concerns, use the GitHub Security tab so the report stays private.

## Scope

This code of conduct applies to project spaces, issue threads, pull requests, releases, and any other venue where PACX maintainers represent the project.

The full Contributor Covenant text is available at https://www.contributor-covenant.org/version/2/1/code_of_conduct.html.
