# Repository Settings Note

The following GitHub repository settings still need to be enabled manually for the CI/CD hardening track:

- Secret scanning
- Push protection
- Dependency graph
- Dependabot security alerts only

Notes:

- These settings are repository-level configuration, not code changes.
- Version-update Dependabot pull requests remain out of scope for this track.
- Security alerts should be enabled without creating extra maintenance noise from version-update PRs or email digests.
