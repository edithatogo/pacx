# PACX Documentation

PACX is a Power Platform CLI extension focused on Dataverse, solution automation, AI Builder, custom connectors, governance, and developer workflows.

## Start Here

- Install the pinned .NET SDK from `global.json`.
- Restore and build from the repository root.
- Configure authentication before running Dataverse or Power Platform commands.
- Use the command reference for command discovery and the recipes for common workflows.

```powershell
dotnet restore
dotnet build
dotnet test
```

## Documentation Sections

- **Authentication**: supported local and automation-friendly authentication flows.
- **Commands**: generated and curated command reference.
- **Recipes**: scenario-based guides for common tasks.
- **ADRs**: architectural decision records for maintainers.
- **Upgrade Guide**: mapping notes for users moving from `pac` to `pacx`.
