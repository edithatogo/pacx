# Browse Ecosystems

Use the catalog-backed browse commands to inspect adjacent tooling and package providers without leaving the CLI.

```powershell
pacx tool browse
pacx tool browse --query dataverse
pacx tool run --name Dataverse MCP
pacx package source browse
pacx package source browse --query power bi
```

This is useful when you want to compare PACX commands against nearby ecosystems, identify missing capabilities, or jump to the upstream source for a package or tool.
