# Flow Studio Capability Workflow

Use this workflow when you need to explore or inspect Flow Studio-style flow operations from PACX.

```powershell
pacx mcp flow browse
pacx mcp flow browse --category Debug
pacx mcp flow inspect --name "Flow Studio Debug"
pacx mcp flow monitor
pacx mcp flow run --name "Flow Studio Authoring"
pacx mcp flow debug
pacx mcp flow govern
```

These commands are useful when you are mapping agent-driven flow workflows, comparing potential PACX surfaces, or checking the catalog shape before adding more automation.
