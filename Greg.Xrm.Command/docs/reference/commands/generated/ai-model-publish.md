# ai model publish

Publish a trained AI Builder model to an environment.

## Usage

```powershell
pacx ai model publish
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--model-id` | m | string | True | AI Builder model ID to publish. |
| `--dry-run` |  | bool | False | Show what would be published without actually publishing. |
