# column seed

Sets the seed for a given column. Is a proxy for [SetAutoNumberSeed Action](https://learn.microsoft.com/en-us/power-apps/developer/data-platform/webapi/reference/setautonumberseed?view=dataverse-latest)

## Usage

```powershell
pacx column seed
```
