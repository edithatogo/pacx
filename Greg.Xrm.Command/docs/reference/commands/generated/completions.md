# completions

Print a shell completion script for pacx.

## Usage

```powershell
pacx completions
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--shell` | s | CompletionShell | False | Shell to generate completions for: pwsh, bash, zsh, fish. |
