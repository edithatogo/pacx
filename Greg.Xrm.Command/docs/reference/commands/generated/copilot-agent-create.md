# copilot agent create

Create a Copilot Studio agent.

## Usage

```powershell
pacx copilot agent create
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--name` | n | string | True | Agent display name. |
| `--solution` | s | string? | False | Solution ID or unique name. |
