# dataset list

List Power BI datasets in a workspace.

## Usage

```powershell
pacx dataset list
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--workspace-id` | w | string | True | Power BI workspace ID. |
