# diag

Run diagnostics to troubleshoot PACX setup.

## Usage

```powershell
pacx diag
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--verbose` | v | bool | False | Show detailed diagnostic information. |
