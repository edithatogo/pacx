# fabric workspace create

Create a Microsoft Fabric workspace.

## Usage

```powershell
pacx fabric workspace create
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--name` | n | string | True | Workspace display name. |
| `--capacity-id` |  | string? | False | Optional Fabric capacity ID. |
