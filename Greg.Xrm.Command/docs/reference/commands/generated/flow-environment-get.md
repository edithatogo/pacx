# flow environment get

Get details of a Power Automate environment.

## Usage

```powershell
pacx flow environment get --environment <name>
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--environment` | env | string | True | The environment name or ID. |
