# flow recyclebin list

List soft-deleted (recycle bin) Power Automate cloud flows.

## Usage

```powershell
pacx flow recyclebin list --environment <name>
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--environment` | env | string | True | The environment name or ID. |
