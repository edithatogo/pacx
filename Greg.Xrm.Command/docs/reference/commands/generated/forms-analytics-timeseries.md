# forms analytics timeseries

Export Microsoft Forms response analytics as a time series.

## Usage

```powershell
pacx forms analytics timeseries
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--bucket` | b | string | False | Time bucket: day or week. |
