# forms duplicate

Duplicate/clone a Microsoft Form.

## Usage

```powershell
pacx forms duplicate
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--new-title` | n | string? | False | Title for the duplicated form. Defaults to original title + ' (Copy)'. |
