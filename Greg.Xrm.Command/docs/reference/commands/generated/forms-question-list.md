# forms question list

List questions in a Microsoft Forms authoring manifest.

## Usage

```powershell
pacx forms question list --file <path>
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--file` | f | string | True | Path to the authoring manifest JSON. |
