# forms response get

Get a single response from a Microsoft Form.

## Usage

```powershell
pacx forms response get
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--response-id` | r | int | True | The response ID to retrieve. |
