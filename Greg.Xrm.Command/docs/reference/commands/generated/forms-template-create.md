# forms template create

Create a Microsoft Forms template from a form.

## Usage

```powershell
pacx forms template create
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--from-form` | f | string | True | Source form ID. |
| `--scope` | s | string | False | Template scope: org or team. |
