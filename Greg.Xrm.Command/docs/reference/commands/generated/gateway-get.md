# gateway get

Get details of a specific on-premises data gateway.

## Usage

```powershell
pacx gateway get
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--id` | i | string | True | The ID of the gateway to retrieve. |
