# history get

Get the list of commands executed in the past

## Usage

```powershell
pacx history get
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--file` | f | string? | False | If you want to save the command list to a specific file, specify the name of the file. |
