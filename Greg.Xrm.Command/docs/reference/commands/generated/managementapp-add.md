# managementapp add

Add a Power Platform management application.

## Usage

```powershell
pacx managementapp add
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--application-id` | i | string | True | The application (client) ID of the Azure AD app registration. |
