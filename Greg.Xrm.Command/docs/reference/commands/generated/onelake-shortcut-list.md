# onelake shortcut list

List OneLake shortcuts for a Fabric item.

## Usage

```powershell
pacx onelake shortcut list
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--workspace-id` | w | string | True | Fabric workspace ID. |
| `--item-id` | i | string | True | Fabric item ID. |
