# package remove solution

Remove a solution payload from a PACX package folder.

## Usage

```powershell
pacx package remove solution
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--force` | f | bool | False | Remove the manifest entries even if the artifact file is missing. |
