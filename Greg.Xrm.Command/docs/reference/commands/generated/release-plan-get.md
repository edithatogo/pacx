# release-plan get

Get details of a specific release plan item by ID.

## Usage

```powershell
pacx release-plan get
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--id` | i | string | True | Release plan item ID. |
