# solution component removeAll

Removes all solution components from an unmanaged solution.

## Usage

```powershell
pacx solution component removeAll
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--solution` | s | string? | False | The unique name of the solution. If not provided, the default solution will be used. |
