# column getDependencies

Retrieves the list of solution components that depend from a given column

## Usage

```powershell
pacx column getDependencies
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--forDelete` | d | bool | False | Specifies whether to retrieve the dependencies for delete or not |
