# package deploy

Deploy a PACX-native package to the current Dataverse environment.

## Usage

```powershell
pacx package deploy
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--dry-run` |  | bool | False | Validate the package and print the deployment plan without making changes. |
