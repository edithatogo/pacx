# workflow run list

Lists the recent runs for a specific Flow.

## Usage

```powershell
pacx workflow run list
```

## Options

| Option | Short | Type | Required | Description |
| --- | --- | --- | --- | --- |
| `--limit` | l | int | False | The maximum number of runs to return (default: 10). |
